# Karate Bot
An open-source bot for Roblox Karate.

## Features
- Autoblock
- Attacks after they block (If they panic block it punishes them)
    - Works rlly well on styles that can't hold blocks for long
- Strifes left and right
- Follows the person
- Has delays for blocks and hits
- Lock on
- Flicks before hitting
- (Some extra ones I forgot)

## How to Use
- Use Wado Ryu for best results
- Use default settings
- Make sure shift lock is unlocked (For flicks)
- F to enable/disable fighting
- R to toggle the guis visibility
